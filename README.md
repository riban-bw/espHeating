# espHeating
Home heating controller based on ESP8266 micro-controller

A hot water and space (radiator pump) controller running on the ESP8266 micro-controller.
Provides:

* Timed events
* WiFi remote control
* Manual override
* Temperature set point
* Multiple temperature sensors (DS1820 1-wire)
